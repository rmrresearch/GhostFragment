Detailed Description

Describes how a supersystem has been broken up into unions of fragments.

This class models a view of a supersystem after it has been decomposed into unions of fragments, i.e., n-mers. The n-mers remember their history (i.e., know which fragments formed them) and the fragments are accessible as well.

This function will flatten the provided n-mer into a set of atoms (the actual process simply takes the union of the atoms in each fragment and thus works for non-disjoint n-mers and fragments) and then takes the union of the AO basis sets for each atom in the resulting set.
