Detailed Description

Namespace collecting all of the property types used throughout GhostFragment.

GhostFragment uses a number of property types. To protect the source code of GhostFragment from upstream changes to those property types, and for consistency, we typedef the property types we use in the pt namespace and rely on the typedefs throughout GhostFragment.

ghostfragment::pt::DECLARE_PROPERTY_TYPE ( MolecularGraph  )

Property type for forming a molecular graph.

There are a number of use cases in computational chemistry which lead to the molecular system being thought of as a mathematical graph. Modules which satisfy this property type are responsible for taking a fragmented molecule (fragments define the nodes of the graph) and creating a molecular graph from it (essentially the module must assign connectivity to the nodes).
