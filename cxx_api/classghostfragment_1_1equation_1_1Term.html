Detailed Description

Desribes a term in an expression.

Terms map to individual calculations. Superficially a Term is just a ChemicalSystem and a coefficient for scaling the result.

bool ghostfragment::equation::Term::operator== ( const Term &  rhs) const
noexcept

Compares *this to rhs for value equality.

Two Term instances are value equal if they are both empty, or if they both contain the same state. In comparing state the molecular system, AO basis set, and scaling coefficient are compared. N.B. all floating point comparisons are also value equality (i.e., floating point values are compared to machine epsilon).
