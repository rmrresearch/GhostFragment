Detailed Description

A set of terms.

Expressions describe how to combine Term objects to compute a property of interest.

◆ add_term()

void ghostfragment::equation::Expression::add_term ( term_type  term)

Adds term to *this.

This method behaves similar to push_back in that Expression behaves like a random access container and that term will be added on to the end of *this. This method does not check if term is already in *this.

Parameters
[in]termThe piece of the linear expansion to add.
Exceptions
std::bad_allocif there is a problem adding term to *this. Strong throw guarantee.
