Weighted Distance

Weighted Distance Capper

This module closes off the valencies of the input fragments using a hydrogen atom at a specific distance. More specifically for each bond A-B, such that atom A is in the fragment, and atom B is not, a hydrogen atom will be added to the fragment such that the location along the original bond is the ratio of a normal A-H bond to a normal A-B bond.
