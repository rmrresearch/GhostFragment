Developer Documentation
#######################

The purpose of these topics are to explain how the internals of GhostFragment,
and to provide assistance to developers wanting to contribute to GhostFragment.
The GhostFragment project has adopted the same development solutions as the
NWChemEx project and developers are encouraged to also peruse the development
topics available `here <https://nwchemex-project.github.io/.github/>`__.
